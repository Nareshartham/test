Abstracting Code with Functions


We can now apply abstraction and convert this into a function that can compute the area of the given shape for any x and y. 

def shape_area(x, y):
    return sq_area(x) + triangle_area(x, y)


Note the order of arguments x and y in the call to the triangle_area. We had written triangle_area with the expectation that the first parameter would stand for the base and the second for the height. Since x is used in calling sq_area within the function body, we know that it stands for the base of the triangle.
